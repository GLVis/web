GLVis is a lightweight tool for accurate and flexible finite element visualization.

Features

  • Accurate functional representation of many finite elements.
  • Support for arbitrary high-order and NURBS meshes.
  • Server mode accepting multiple socket connections.
  • Visualization of parallel meshes and solutions.
  • ... and many more.

GLVis is based on the MFEM library and is currently used in the BLAST, hypre and XBraid projects. See also our Gallery.
